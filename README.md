# 图书管理系统 java1100

## 项目概述

本图书管理系统是基于前后端分离的架构设计，使用 Spring Boot 和 Vue.js 作为主要技术栈。系统支持普通用户、管理员、超级管理员等多种角色，功能全面，涵盖了图书信息管理、用户借阅处理等日常图书管理业务需求。

## 技术栈

- **系统环境**：Windows
- **开发环境**：
  - IDEA
  - Maven
  - Mysql 8.0
- **技术栈**：
  - Java
  - Mysql
  - Spring Boot
  - Vue.js

## 功能模块

- AI助手
- 个人主页
- 图书信息管理
- 图书类别管理
- 借阅信息管理
- 用户管理
- 修改密码
- 用户借书
- 用户还书

## 系统角色

- 普通用户
- 管理员
- 超级管理员

## 运行环境

- **后端**：Java 1.8+，Mysql 8.0+
- **前端**：现代浏览器支持（推荐使用 Chrome 或 Firefox）

## 目录结构

```
项目目录结构
|-- 前端代码（Vue.js）
|-- 后端代码（Spring Boot）
    |-- src
        |-- main
            |-- java
            |-- resources
|-- 数据库脚本
|-- 部署文档
```

## 部署步骤

1. 环境准备：确保 Java、Mysql 等依赖软件已正确安装。
2. 数据库搭建：执行数据库脚本，建立项目所需数据库及表。
3. 后端部署：配置 Spring Boot 项目，编译打包后运行。
4. 前端部署：编译 Vue.js 项目，部署到 Web 服务器。

## 核心亮点

- 前后端分离设计，便于维护与扩展。
- 采用 Spring Boot 和 Vue.js 现代化技术栈，开发效率高。
